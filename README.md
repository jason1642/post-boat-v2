# Post Boat 

This project allows users to post content and send private messages. Created via MERN stack.

## Installation
```
npm install
npm start
nodemon api/server.mjs 
```

### Important Dependencies
- Socket.io
- Redux
- Material.ui


### `npm test`
To test back end routing. Must be able to connect to database.
```
npm run test
```

### Dotenv variables
```
MONGODB_USERNAME
MONGODB_PASSWORD
TOKEN_SECRET
```
